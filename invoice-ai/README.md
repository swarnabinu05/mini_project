# Invoice AI System

An intelligent invoice processing system that uses OCR to read invoices, validates data against business rules, and automatically updates company databases.

## Features

- OCR-based invoice text extraction
- Automated data validation (tax rules, country regulations)
- Error detection and highlighting
- Digital signature application
- Database integration
- Manual review workflow for flagged invoices

## Getting Started

1. Backend setup: `cd backend && pip install -r requirements.txt`
2. Frontend setup: `cd frontend && npm install`
3. Start backend: `cd backend && python app/main.py`
4. Start frontend: `cd frontend && npm start`
